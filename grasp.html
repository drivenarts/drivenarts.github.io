Overview

GRASP SDK is a powerful Unreal Engine plugin that enables developers to program and control humanoid robots using familiar Unreal Engine tools and workflows. Built on the foundation of VR training experiences, GRASP brings the power of game development to robotics.

GRASP SDK powering service automation at Donut Haven

High-Level Tasks

Focus on what needs to be done using familiar Unreal tools, while the SDK handles the complexities of robot execution.

Familiar Tools

Use Blueprints and C++ with the same workflows you know, making robot programming as intuitive as game development.

Robot Integration

Seamlessly integrate with leading robot platforms while focusing on business logic rather than low-level control.

C++ API

High-Level Robot Control API

#pragma once

#include "CoreMinimal.h"
#include "Kismet/BlueprintFunctionLibrary.h"
#include "GraspBlueprintFunctionLibrary.generated.h"

UENUM(BlueprintType)
enum class EGraspHand : uint8
{
    Left,
    Right
};

UENUM(BlueprintType)
enum class EGraspStyle : uint8
{
    Pinch,
    FullGrip,
    Custom
};

UENUM(BlueprintType)
enum class EGraspInteractionType : uint8
{
    None,
    Pull,
    Push,
    Rotate,
    Press,
    Custom
};

UCLASS(Blueprintable)
class DUNKIN_API UGraspBlueprintLibrary : public UBlueprintFunctionLibrary
{
    GENERATED_BODY()

public:

    UFUNCTION(BlueprintCallable, BlueprintNativeEvent, Category = "Grasp|Actions")
    bool MoveRobotTo(
        AActor* RobotActor,
        const FVector& Destination,
        const FRotator& Orientation,
        float Speed = 1.0f
    );

    UFUNCTION(BlueprintCallable, BlueprintNativeEvent, Category = "Grasp|Actions")
    bool MoveHandTo(
        AActor* RobotActor,
        EGraspHand Hand,
        const FVector& Destination,
        const FRotator& Orientation,
        float Speed = 1.0f
    );

    UFUNCTION(BlueprintCallable, BlueprintNativeEvent, Category = "Grasp|Actions")
    bool PickUpObject(
        AActor* RobotActor,
        EGraspHand Hand,
        AActor* TargetObject,
        EGraspStyle GripStyle
    );

    UFUNCTION(BlueprintCallable, BlueprintNativeEvent, Category = "Grasp|Actions")
    bool PlaceObject(
        AActor* RobotActor,
        EGraspHand Hand,
        AActor* TargetObject,
        const FVector& Location
    );

    UFUNCTION(BlueprintCallable, BlueprintNativeEvent, Category = "Grasp|Actions")
    bool UseObject(
        AActor* RobotActor,
        EGraspHand Hand,
        AActor* TargetObject,
        EGraspInteractionType Interaction
    );
};

Key Concepts

  • Blueprint Integration: All core functionality is exposed to Blueprints for visual programming.
  • Intuitive Controls: Simple, high-level functions for common robot actions like movement, grasping, and object interaction.
  • Flexible Interaction: Support for different grip styles and interaction types to handle various scenarios.
  • Hand-Specific Control: Independent control of left and right robot hands for complex manipulation tasks.
